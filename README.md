# HireSense AI

HireSense AI is an advanced resume intelligence web application built with Python and Streamlit. It helps users analyze resumes, calculate ATS scores, detect skill gaps, screen multiple candidates, and generate ATS-friendly resume drafts with AI-assisted writing support.

## Features

- Resume Analyzer with ATS score
- Skill Gap Analyzer by job role
- Bulk Resume Screening
- CSV-based candidate ranking
- ATS Resume Builder
- AI-assisted resume content generation
- Offline resume builder fallback
- Role-wise skill suggestions
- PDF report download
- Designed resume PDF export
- Dark mode support

## Tech Stack
